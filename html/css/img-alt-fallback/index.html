img 読み込み失敗時の alt 表示テスト

すべて同じ未存在画像を参照し、alt テキストの見え方のみを比較します。img 要素の表示サイズは固定です。

1. ブラウザのデフォルトスタイル

img のサイズ固定以外は未指定。

2. リンク内 img のデフォルトスタイル

a 要素の中に配置したケース。

3. border: none

img の枠を明示的に除去したケース。

4. font 関連を img に直接指定

font-family / font-size / font-weight / line-height を img に指定。

5. 親要素から font 関連を継承

font 関連プロパティは親要素に設定し、img 側は未指定。
