overflow-clip-margin のテスト

.box {
	width: 200px;
	height: 100px;
	padding: 20px;
	overflow: clip;
	overflow-clip-margin: 40px;
	border: solid 1px #888;
	background: #feeec0;
	background-clip: content-box;
}
